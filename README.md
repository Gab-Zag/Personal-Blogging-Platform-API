# Personal Blogging Platform API

## Descrição

Uma API RESTful para gerenciamento de um blog pessoal, desenvolvida como projeto educacional para aprofundar conhecimentos em desenvolvimento backend.

## Objetivo

Este projeto foi criado para estudos com fins de:
- Aprender conceitos de desenvolvimento de APIs REST
- Praticar boas práticas de programação
- Implementar autenticação e autorização
- Trabalhar com bancos de dados
- Documentar código profissionalmente

## Funcionalidades

- Criar, ler, atualizar e deletar posts

## Tecnologias

- Java
- Springboot
- PostgreSQL
- Postman
- HTML
- CSS
- JavaScript

## Contribuição

Projeto pessoal para fins educacionais.
